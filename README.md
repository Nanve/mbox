## mbox.js


mBox是为移动设备（手机、平板等webkit内核浏览器/webview）量身定做的弹层插件，插件采用JavaScript编写，您需要按照场景选择使用。插件开源、体积小、简单实用、功能强大，是你架构移动终端网站的重要选择！
